# vrtx-studio — Creative Agency Frontend

A high-performance creative agency site built with Next.js and modern design principles.

## 🚀 Key Features
- **Modern Stack:** Built with Next.js for elite performance and SEO.
- **Agency Aesthetic:** Refined typography and sophisticated layouts.
- **Fully Responsive:** Optimized for all screen sizes from mobile to wide monitors.
- **Dynamic Content:** Structured to handle portfolio showcases and studio services.

## 🛠️ Tech
- Next.js (App Router)
- React
- Tailwind CSS
- Framer Motion
